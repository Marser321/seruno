# Landing de Ventas — SER UNO

Repositorio de despliegue de la página de compra directa de **SER UNO**, el programa principal de la Psicóloga Gisella Arias Olson.

La fuente editable vive en `Skill Landings/ecosistema/eco-ser`. Este repositorio contiene la compilación estática generada con Vite para publicar en Vercel.

## Estructura de conversión

1. Hero tradicional con H1, subtítulo y CTA visible en el primer pantallazo.
2. Planes e inversión inmediatamente después del hero.
3. Contenido del programa.
4. Carrusel lateral con nueve testimonios en video.
5. Preguntas frecuentes y CTA final.

Los originales en 1080p se conservan fuera del repositorio. Las versiones web están optimizadas a 720p y usan portadas livianas y carga diferida.
